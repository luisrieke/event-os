# EventOS

Find the one event worth leaving home for.

Email:

> What should I do tonight in Berlin?

EventOS learns what you like, scans events across the internet, and recommends a single event with the highest expected ROI.

No browsing.
No endless lists.
No FOMO.

Just one recommendation.

### Example

You:
> I'm a developer. I like Cursor.

EventOS:
> Go to Cursor Hackathon meets Builders Night Berlin.

### How it works

- Learns from your feedback
- Builds a profile over time
- Scans and ranks events
- Recommends one event

### Status

Hackathon project. Built in one evening.

### Vision

An AI event scout in your inbox.
